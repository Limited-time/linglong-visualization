# # 灵笼 · 多维世界观建模

基于《灵笼》S1-S2全部伏笔的系统性推演，包含：
- 五维并行时间线
- 人物-阵营关系拓扑
- 25项伏笔回收矩阵
- 世界观维度分层
- idea：
-   灵笼一二季世界观及逐伏笔和回扣整理；
-   将所有伏笔故事圆上，并且多线交织并行；
-   外网二创以及主题曲和插曲里的叙事性&趋势与伏笔.
